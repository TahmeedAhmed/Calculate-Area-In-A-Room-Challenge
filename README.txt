If running on Terminal:
	- Download folder into a directory x

	- run command 'javac CalculateAreaInRoom.java'
	- run command 'java CalculateAreaInRoom'

If running on an IDE:
	- Download folder into a directory x
	- Open project in IDE (File -> Open Project (find x))
